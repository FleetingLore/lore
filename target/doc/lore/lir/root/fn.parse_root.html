pub fn parse_root(raw_lines: Vec<&str>) -> Root<'_>
Expand description

解析多行 LIR 文本，构建完整的文档结构。

这是 LIR 解析器的入口函数，它接收多行文本并返回结构化的文档表示。

返回 Root 结构，包含解析后的所有行信息。

use crate::lir::parse_root;

let lines = vec![
    "root element",
    "  + domain definition",
    "    nested element",
];
let root = parse_root(lines);

// 可以格式化输出整个文档
println!("{}", root);
