§内容模块

每一行的数据，包括缩进级别和内容。 根据是否伴随子域，内容分为要素或领域这两种类型。

以 + 开头，加号后有空格，则解析为 Content::Domain。 其余其他情况，解析为 Content::Element。

[ * content ]            -> Element("[ * content ]")
+ [ * content ]          -> Domain("+ [ * content ]")
+[ * content ]           -> Element("+[ * content ]")     // 无空格。
+                        -> Element("+")          // 无空格和内容。
+                        -> Domain("")            // 有空格但无内容。
